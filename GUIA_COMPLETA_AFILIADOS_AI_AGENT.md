# 3. LOS 6 PILARES DE LA ESTRATEGIA

## Pilar 1: 🎯 Fundamentos y Autoridad
*"Posiciónate como experto sin desesperación"*

**Pasos:**
1. Define tu nicho de mercado específico
2. Investiga los problemas y deseos de tu audiencia
3. Crea contenido que demuestre expertise sin vender
4. Desarrolla habilidades de persuasión y comunicación
5. Estudia a la competencia exitosa en tu nicho

⚠️ *No caigas en la desesperación por vender. La autoridad se construye con tiempo.*

## Pilar 2: 📄 Fan Page Profesional
*"Construye tu marca en Facebook"*

**Pasos:**
1. Crea una Fan Page con nombre profesional relacionado a tu nicho
2. Diseña logo y portada profesional (usa Canva)
3. Completa toda la información de la página
4. Publica 5-10 posts de valor antes de promocionar
5. Invita a amigos y conocidos a dar like

⚠️ *Nunca uses tu perfil personal para promocionar. La Fan Page te protege.*

## Pilar 3: 🔍 Embudo de Grupos
*"Filtra y selecciona los grupos adecuados"*

**Pasos:**
1. Busca grupos en Facebook relacionados a tu nicho
2. Usa palabras clave estratégicas
3. Únete a 20-30 grupos como mínimo
4. Observa las reglas de cada grupo
5. Publica contenido de valor primero en cada grupo
6. Filtra: quédate solo con los que funcionan

⚠️ *No publiques en todos los grupos a la vez. Prueba cada grupo con contenido de valor primero.*

## Pilar 4: 📊 Contenido 80/20
*"80% valor, 20% promoción"*

**Pasos:**
1. Planifica 10 contenidos: 8 de valor, 2 promocionales
2. Contenido de valor: tips, tutoriales, casos de éxito, tendencias
3. Contenido promocional: reseñas honestas, comparativas, ofertas
4. Usa un calendario semanal para organizar las publicaciones
5. Mide el engagement de cada tipo de contenido

⚠️ *Si solo promocionas, la comunidad te rechazará. Respeta el 80/20.*

## Pilar 5: 💬 Conversión WhatsApp
*"Captura leads y cierra ventas"*

**Pasos:**
1. Configura tu número de WhatsApp Business
2. Genera enlaces wa.me: `https://wa.me/tu-numero`
3. Incluye el enlace en tu biografía de la Fan Page
4. En posts promocionales: "Comenta 'INFO' y te envío detalles por WhatsApp"
5. Prepara un guión de ventas para la conversación
6. Da seguimiento personalizado a cada lead

⚠️ *No envíes spam por WhatsApp. Cada lead merece atención personalizada.*

## Pilar 6: ⚡ Acción y Constancia
*"Interacción diaria: mañana, tarde y noche"*

**Pasos:**
1. Establece 3 momentos de interacción: mañana, tarde, noche
2. Mañana: comparte contenido de valor en 3-5 grupos
3. Tarde: responde comentarios e interactúa en los grupos
4. Noche: revisa métricas y planea el día siguiente
5. Mantén constancia por 21 días para crear el hábito

⚠️ *La clave no es la intensidad, es la constancia. Mejor poco todos los días.*

# 4. MANUAL DE INSTALACIÓN

## Requisitos del Sistema

- **Python**: 3.8 o superior
- **Sistema Operativo**: Windows, macOS, Linux
- **Conexión a Internet**: Para API y dashboard web
- **Navegador**: Chrome, Firefox, Edge (para el dashboard)
- **Google Chrome**: Necesario para Facebook Poster (SeleniumBase)

## Instalación Paso a Paso

### Paso 1: Clonar o descargar el proyecto

```bash
# Si tienes git:
git clone https://github.com/tu-usuario/affiliate-ai-agent.git
cd affiliate-ai-agent

# O simplemente navega a la carpeta del proyecto
cd affiliate-ai-agent
```

### Paso 2: Instalar dependencias base

```bash
pip install -r requirements.txt
```

Esto instalará:
- `rich>=13.0.0` — Terminal interactiva con colores y paneles
- `flask>=3.0.0` — Servidor web del dashboard
- `flask-cors>=4.0.0` — CORS para el dashboard
- `requests>=2.31.0` — Peticiones HTTP
- `schedule>=1.2.0` — Planificador de rutinas
- `colorama>=0.4.6` — Colores en terminal

### Paso 3: Instalar Facebook Poster (Opcional pero recomendado)

```bash
pip install seleniumbase
```

SeleniumBase (~14MB) permite la automatización real con modo indetectable.

### Paso 4: Verificar la instalación

```bash
python -c "from agent.database import Database; from agent.strategy import StrategyEngine; from agent.knowledge_base import KnowledgeBase; from agent.group_finder import GroupFinder; from agent.content_planner import ContentPlanner; from agent.lead_tracker import LeadTracker; from agent.campaign import CampaignManager; from automation.facebook_poster import FacebookPoster; print('✅ TODOS LOS MÓDULOS INSTALADOS CORRECTAMENTE')"
```

### Paso 5: ¡Ejecutar!

```bash

python run.py

python run.py --dashboard
```

# 8. MÓDULOS DEL AGENTE

## 📦 strategy.py — Motor de Estrategia

**Propósito**: Corazón del sistema. Implementa los 6 pilares y orquesta todos los demás módulos.

**Clases principales**:
- `Config`: Configuración global (nicho, fan page, whatsapp)
- `Campaign`: Modelo de campaña con grupos, contenido y leads
- `Group`: Modelo de grupo de Facebook
- `ContentItem`: Modelo de contenido (valor o promoción)
- `Lead`: Modelo de lead/contacto
- `StrategyEngine`: Motor principal con métodos para gestionar todo

**Métodos clave**:
```python
engine = StrategyEngine()
engine.create_campaign("Mi Campaña", niche="finanzas")
engine.get_pillar_guidance(3)  # Guía del pilar 3
engine.export_campaign_report("Mi Campaña")
```

## 🧠 knowledge_base.py — Base de Conocimiento

**Propósito**: Almacena todo el conocimiento experto: psicología, frases, guiones, nichos.

**Contenido**:
- `SALES_PSYCHOLOGY_TIPS`: 8 principios de persuasión
- `ENGAGEMENT_PHRASES`: Frases de gancho, CTA y cierre
- `VALUE_CONTENT_TYPES`: 8 tipos de contenido de valor
- `GROUP_WARMUP_TIPS`: 5 pasos para activar grupos
- `WHATSAPP_SCRIPT`: Guión completo: saludo → descubrimiento → objeciones → cierre
- `PROFITABLE_NICHES`: 8 nichos con nivel de demanda
- `KEY_METRICS`: 6 KPIs para medir éxito

## 🔍 group_finder.py — Buscador de Grupos

**Propósito**: Gestiona el embudo de descubrimiento y filtrado de grupos.

**Características**:
- Palabras clave por nicho para buscar en Facebook
- Pipeline de 5 estados: discovered → pending_test → testing → active → rejected
- Checklist de calidad (8 criterios)
- Sugerencias inteligentes de acción

## 📊 content_planner.py — Planificador 80/20

**Propósito**: Genera y organiza contenido con la regla 80/20.

3. La prueba social vende ("más de 5000 estudiantes satisfechos")

8. Las objeciones son oportunidades
